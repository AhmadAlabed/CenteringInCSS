Centering in CSS



1- Horizontally


A) To Center Text or Links : .center { text-align: center; }


B) To Center Block-Level Elements Like a div : .center { margin: 0 auto; }


C) To Center a Div Horizontally with Flexbox : .parent { display: flex; justify-content: center; }


2- Vertically


A) To Center a Div Vertically with CSS Absolute Positioning and Negative Margins : .parent { position: relative; } .child { position: absolute; top: 50%; margin-top: -25px;/* Half this element's height */ }


B) To Center a Div Vertically with CSS Absolute Positioning and Transform : .parent { position: relative; } .child { position: absolute; top: 50%; transform: translate(0, -50%); }


C) To Center a Div Vertically with Flexbox : .parent { display: flex; align-items: center; }


3- Both Horizontally & Vertically


A) To Center a Div Vertically and Horizontally with CSS Absolute Positioning and Negative Margins : .parent { position: relative; } .child { width: 50px; height: 50px; position: absolute; top: 50%; left: 50%; margin: -25px 0 0 -25px; }


B) To Center a Div Vertically and Horizontally with CSS Absolute Positioning and Transform : .parent { position: relative; } .child { position: absolute; top: 50%; left: 50%; transform: translate(-50%, -50%); }


C) To Center a Div Vertically and Horizontally with Flexbox : .parent { display: flex; justify-content: center; align-items: center; }
